טיפ: .gitignore

קובץ .gitignore מגדיר אילו קבצים Git יתעלם מהם לחלוטין — סיסמאות, קבצי cache, לוגים. צור אותו בשורש הפרויקט לפני git add .

⚠️

אזהרה: סיסמאות

קובץ .env מכיל סיסמאות ומפתחות API. אסור להעלות אותו ל-GitHub. וודא שהוא מופיע ב-.gitignore לפני כל push.

טבלת פקודות מהירה
פעולה רמה פקודה
בדוק מצב נוכחי בסיסי git status
הוסף הכל ל-staging בסיסי git add .
צור commit בסיסי git commit -m "message"
העלה ל-GitHub בסיסי git push
משוך מ-GitHub בסיסי git pull
הצג היסטוריית commits מידע git log --oneline
הצג שינויים מפורטים מידע git diff
מחק קובץ מ-Git ודיסק מחיקה git rm filename
בטל שינויים בקובץ ביטול git checkout -- filename
קבע כתובת remote הגדרה git remote add origin URL
